## Week 8 - HTML, CSS, JavaScript
Nessa aula o David inicia falando sobre como funciona a internet e sobre alguns protocolos que garantem o seu funcionamento. Iniciando por **TCP/IP** (Transmission Control Protocol / Internet Protocol), ele usa exemplos visuais e concretos com cartas para mostrar o papel desse conjunto de protocolos. Na rede, entidades (como computadores e servidores) possuem um **IP** para serem identificadas. Uma entidade envia um pacote de dados para outra. Esse pacote de dados passa por outras entidades na rede até encontrar o seu destino. Um computador tentando acessar o Google envia um pacote de dados com a requisição para o servidor que depois retorna uma resposta com os dados para carregar a página de buscas.

Essa introdução aos protocolos segue para explicar **DNS**, **DHCP**, **rotas**, **HTTP** e **HTTPS**. Cada explicação vai se conectando com a anterior para formar uma visão completa de como uma aplicação web funciona. Através dessa explicação ele mostra as ferramentas do DevTools no Chrome para inspecionar os elementos da página e visualizar os **status** das requisições feitas para acessar uma página.   

Na explicação sobre o desenvolvimento de uma página, o David destaca que o **HTML**  é uma linguagem de marcação e não é uma linguagem de programação. Com essa linguagem a estrutura da página pode ser construída usando **tags** com **propriedades**, que seguem a estrutura de chave e valor. É muito bom como ele continua fazendo paralelos com as apresentações anteriores sobre **C**, **estruturas de dados** e **Python**. 

Usando a tag `<style>` ele mostra como aplicar propriedades de estilização em elementos **HTML**. Nisso ele apresenta diferentes pontos em que a estilização de elementos pode ser aplicado em uma página, usando inline, interno com a tag `<style>` e importando o arquivo **CSS**. Falando sobre **CSS** ele também mostra seletores por tag, id ou classe e como os elementos podem ser estilizados de diferentes formas.

Chegando na explicação do **JavaScript** é apresentado como essa linguagem de programação pode ser usada em arquivos **HTML** e como DOM (Documento Object Model) é usado para aplicar dinamismo na página.|
